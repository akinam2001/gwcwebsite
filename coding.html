Our Focus

Languages

We use Scratch to teach the basics of coding. The format is easy to follow and has no complicated syntax. It is a fun, visual way to learn variables, loops, conditionals, and functions.

HTML is a popular language and is fun to implement. It creates easy and quick projects but can easily turn into an in-depth task. We use it as a way to teach organization of code as well as proper coding etiqutte.

CSS is used in pair with HTML. We use it add depth to our HTML and start the transition into simple JavaScript. CSS adds to the organization of code and also shows how everything fits in a program.
